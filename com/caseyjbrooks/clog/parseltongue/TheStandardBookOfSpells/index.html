public static Object ansiForegroundColorStart(Object, String)
Method Annotations:
@Spell(name="fg")

Inserts the appropriate ANSI codes to change the foreground color of the text. The foreground color will be used until an ANSI 'reset' character is added with the 'reset' Spell. Should only be used when output is intended to be displayed in a Terminal. Works on both Unix/Linux terminals and Windows command prompt. Available Colors (case insensitive): BLACK, RED, GREEN, YELLOW, BLUE, MAGENTA, CYAN, WHITE, DEFAULT Clog example: #{ $0 | fg('cyan') }
